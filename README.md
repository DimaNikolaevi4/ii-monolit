# ИИ-Монолит

Лендинг сайт для ИИ-Монолит — комплексные решения на базе искусственного интеллекта.

## Страницы

| Файл | Описание |
|------|----------|
| `index.html` | Главная страница |
| `datasets.html` | Создание датасетов |
| `deployment.html` | Развёртывание моделей |
| `finetuning.html` | Дообучение ИИ |
| `portfolio.html` | Портфолио и экспертиза |
| `legal.html` | Правовая информация |
